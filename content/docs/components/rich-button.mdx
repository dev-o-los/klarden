---
title: Rich Button
description: A highly customizable button with premium 3D effects and color variants.
category: Components
---

The RichButton is a tactile trigger designed for modern interfaces. It features a subtle 3D lift effect, responsive spring physics, and a wide palette of premium color variants.

## Preview

<ComponentPreview name="rich-button">
```tsx
import { RichButton } from "@/components/rich-button";
import { Zap } from "lucide-react";

export default function Example() {
  return (
    <RichButton color="blue" size="lg">
      <Zap className="mr-2 h-5 w-5" />
      Get Started
    </RichButton>
  );
}
```
</ComponentPreview>

## Usage

### Installation

<InstallBlock command="rich-button" />

```tsx
import { RichButton } from "@/components/rich-button";

export default function App() {
  return <RichButton color="emerald">Confirm Action</RichButton>;
}
```
