The LabelInput is a minimalist form input component featuring a smooth floating label animation and optional password visibility toggle. Inspired by Spell UI's clean design.

## Preview

<ComponentPreview name="label-input">
```tsx
import { LabelInput } from "@/registry/klarden-ui/label-input";

export default function Example() {
  return (
    <LabelInput
      label="Email Address"
      type="email"
      placeholder="Enter your email"
      ringColor="blue"
    />
  );
}
```
</ComponentPreview>
