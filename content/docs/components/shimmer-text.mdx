---
title: Shimmer Text
description: A premium text component with a continuous shimmer effect flowing within the text itself.
category: Components
---

The ShimmerText component creates a smooth, continuous shimmer effect that flows within the text characters. The shimmer loops infinitely, creating an elegant effect perfect for headings and taglines.

## Preview

<ComponentPreview name="shimmer-text">
```tsx
import React from "react";
import { ShimmerText } from "@/components/shimmer-text";

export default function Example() {
  return (
    <ShimmerText
      variant="default"
      duration={2}
      className="text-4xl md:text-5xl"
    >
      Build beautiful interfaces
    </ShimmerText>
  );
}
```
</ComponentPreview>
