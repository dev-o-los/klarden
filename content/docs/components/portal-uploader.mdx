The PortalUploader is a centerpiece component for file interactions. It activates a rapidly rotating orbital ring when a file is dragged over, providing a "vortex" feel for data absorption.

## Preview

<ComponentPreview name="portal-uploader">
```tsx
import { PortalUploader } from "@/registry/klarden-ui/portal-uploader";

export default function Example() {
  return (
    <PortalUploader 
      onUpload={(file) => console.log("Uploaded:", file.name)} 
    />
  );
}
```
</ComponentPreview>

## Usage

### Installation

<InstallBlock command="shadcn add https://klarden-ui.com/r/portal-uploader.json" />

### Basic Example

```tsx
import { PortalUploader } from "@/registry/klarden-ui/portal-uploader";

export default function App() {
  return (
    <PortalUploader
      accept="image/*"
      onUpload={(file) => alert(`File ${file.name} received`)}
    />
  );
}
```
