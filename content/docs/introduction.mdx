Klarden UI is a collection of high-end, tactile components designed for modern web applications. Every component is built with a focus on **physics-first motion**, **minimalist aesthetics**, and **strict type safety**.

## Philosophy

Our design language is rooted in "Minimalist Neumorphism". We combine clean, sharp borders with subtle, layered shadows to create interfaces that feel physical and responsive to touch and hover.

## Quick Start

### Install Dependencies

Install the core animation and utility libraries.

```bash
pnpm add framer-motion lucide-react clsx tailwind-merge
```

### Configure Tailwind

Add the Klarden UI color palette and animations to your `tailwind.config.ts`.

### Import Components

Start using premium components in your project.

```tsx
import { CommandOrbit } from "@/registry/klarden-ui/command-orbit";
```

## Component Registry

Klarden UI uses a registry-based distribution system. You can add components directly to your project using the shadcn CLI or by copying the source code from these docs.
